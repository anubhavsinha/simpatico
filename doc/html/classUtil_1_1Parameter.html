Util::Parameter Class Reference
Parameter File IO

Detailed Description

A single variable in a parameter file.

Parameter is a base class for objects that read and write the value of a single C++ variable from or to file. Each Parameter has a Label that contains a std::string label that identifies a variable in a parameter file. Instances of subclass also contain a pointer to a C++ variable of the appropriate type.
