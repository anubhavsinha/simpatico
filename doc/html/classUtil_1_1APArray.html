Detailed Description

template<typename Data>
class Util::APArray< Data >

Automatically resizable PArray.

An APArray is a PArray that grows as needed as objects are appended. Like any PArray, it holds pointers to objects, rather than objects. The associated objects are not destroyed when a PArray is deallocated or destroyed.
