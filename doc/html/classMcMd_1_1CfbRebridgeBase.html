McMd::CfbRebridgeBase Class Reference
Monte Carlo Moves

Detailed Description

Base class configuration bias moves internal segment regrowth moves.

Base class for configuration bias Monte Carlo moves that erase and regrow internal segments of linear polymer chains, with fixed endpoints.
