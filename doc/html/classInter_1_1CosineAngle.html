Inter::CosineAngle Class Reference
Angle Interactions

Detailed Description

A three body angle potential, as a function of angle cosine.

This class implements an angle potential: kappa (1 - cos[theta]).

Return bond angle chosen from equilibrium distribution.

This function returns a bond angle chosen from the Boltzmann distribution of angle for bonds of random orientation. The distribution P(theta) of values of the angle theta is proportional to sin(theta)*exp[-beta*phi(theta) ], where phi(theta) is the bond energy.
