McMd::DumpConfig Class Reference

Detailed Description

Periodically dump system configuration to a new file.

This class uses the System::writeConfig() method to write the entire system configuration periodically, at interval read from file. Names of configuration dump files are constructed by concatenating the System outputPrefix, a dumpPrefix, the literal string "config.", and an integer that is incremented after each dump. The dump prefix is a string that is read by the read() method of this class. For example, if System::outputPrefix is "out/" and dumpPrefix is "dump/", the configurations will be written to files "out/dump/config.0", "out/dump/config.1", etc.
