McMd::NvtDpdVvIntegrator Class Reference
Molecular Dynamics Integrators

Detailed Description

A velocity-Verlet DPD integerator.

This class implements a simple velocity-Verlet (VV) algorithm for the dissipitative particle dynamics (DPD) equations of motion.
