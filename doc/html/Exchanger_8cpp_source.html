00118    /*
00119    * void Exchanger::exchangeAtoms()
00120    *
00121    * Exchange ownership of local atoms and groups, set ghost plan.
00122    *
00123    * ------------------------------------------------------------------
00124    * Precondition:
00125    *
00126    * Atomic coordinates must be in scaled [0,1] form on entry.
00127    *
00128    * ------------------------------------------------------------------
00129    * Algorithm:
00130    *
00131    *    - Loop over local atoms, create exchange and ghost communication
00132    *      plans for those beyond or near processor domain boundaries.
00133    *
00134    *    - Add local atoms that will be retained by this processor but
00135    *      sent as ghosts to appropriate ghost send arrays.
00136    *
00137    *    - Call GroupExchanger::markSpanningGroups for each registered
00138    *      GroupExchanger (e.g., bond, angle, dihedral). In this function:
00139    *
00140    *         For each Group<N> {
00141    *            - Set ghost communication flags for groups that span
00142    *              or may span boundaries.
00143    *            - Clear pointers to ghost atoms in the group.
00144    *         }
00145    *      }
00146    *
00147    *    - Clear all ghosts from the AtomStorage
00148    * 
00149    *    - Main loop over transfer directions:
00150    * 
00151    *      For each transfer directions (i and j) {
00152    *
00153    *         For each local atom {
00154    *            if marked for exchange(i, j) {
00155    *               if gridDimension[i] > 1 {
00156    *                  - add to sendAtoms array for removal
00157    *                  - pack into send buffer
00158    *               } else {
00159    *                  shift position to apply periodic b.c.
00160    *               }
00161    *            }
00162    *         }
00163    *
00164    *         if gridDimension[i] > 1 {
00165    *            - Call packGroups for each group type.
00166    *              This packs groups containing atoms that are sent.
00167    *            - Remove exchanged atoms and empty groups
00168    *            - Send and receive data buffers
00169    *            for each atom in the receive buffer {
00170    *               - Unpack atom into AtomStorage
00171    *               - shift periodic boundary conditions
00172    *               - Determine if this is new home (or way station)
00173    *               - If atom is home, add to appropriate ghost arrays.
00174    *            }
00175    *            - Call unpackGroups for each group type.
00176    *         }
00177    *
00178    *      } // end loop over transfer directions
00179    *
00180    *    - Call GroupExchanger::markGhosts each type of group, to create
00181    *      ghost plans for local atoms that must be transferred because
00182    *      they belong to imcomplete groups. Within this function:
00183    *
00184    *         for each Group<N>{
00185    *            if group is incomplete{
00186    *               for each direction (i and j) {
00187    *                  if group is marked for ghost communication {
00188    *                     set ghost flags in plan for local atoms in group 
00189    *                  }
00190    *               }
00191    *            }
00192    *         }
00193    *
00194    * ------------------------------------------------------------------
00195    *  Postconditions: Upon return:
00196    *     Each processor owns all atoms in its domain, and no others.
00197    *     Each processor owns all groups containing one or more local atoms.
00198    *     Ghost plans are set for all local atoms.
00199    *     Send arrays contain local atoms marked for sending as ghosts.
00200    *     The AtomStorage contains no ghost atoms.
00201    * 
00202    * ------------------------------------------------------------------
00203    */
