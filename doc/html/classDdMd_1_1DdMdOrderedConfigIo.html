DdMd::DdMdOrderedConfigIo Class Reference
ConfigIos

Detailed Description

Native / default format for configuration files.

DdMdOrderedConfigIo is a ConfigIo that implements the default configuration file format for ddSim, like DdMdConfigIo, but that outputs atoms in sequential order, sorted by atom id.
