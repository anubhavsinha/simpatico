McMd::EndSwapMove Class Reference
Monte Carlo Moves

Detailed Description

A move that swaps the ends of a linear hetero-polymer.

This move changes the energy if of a linear heteropolymer by reversing the sequence of atom types. When applied to a diblock copolymer, is swaps the A and B blocks. It is pointless if applied to a homopolymer, or to a symmetric ABA triblock.
