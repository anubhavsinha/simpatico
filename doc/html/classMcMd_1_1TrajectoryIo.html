McMd::TrajectoryIo Class Reference
Trajectory File IO

Detailed Description

Trajectory file reader and writer.

A subclass of TrajectoryIo needs to implement a the pair readHeader() and readFrame() or the pair writeHedaer() and writeFrame() or both. If either pair is not implemented, calling the standard implementation will result in an error message.
