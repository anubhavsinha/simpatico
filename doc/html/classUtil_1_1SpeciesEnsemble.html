Util::SpeciesEnsemble Class Reference
Statistical Ensembles

Detailed Description

An ensemble for the number of molecules of one Species.

An SpeciesEnsemble has a type, which can be closed or grand. If the ensemble type is grand, it stores a chemical potential.
