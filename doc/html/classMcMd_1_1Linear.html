McMd::Linear Class Reference
Species

Detailed Description

A Species of linear polymers (abstract).

A linear molecule is a chain of nAtom atoms with consecutive atom indices, with nAtom - 1 bonds between neighboring atoms.

A Linear molecule may be chemically heterogeneous - different atoms and bonds in a chain may be of different types. The chemical structure of a subclass of Linear must be specified by implementing the pure virtual functions atomTypeId() and bondTypeId().

void McMd::Linear::buildLinear ( ) [protected]

Build the chemical structure for a linear molecule.

This method must be called by readSpeciesParam() after values are set for nAtom and nBond. It allocates memory, assigns atom types to all the atoms in a generic molecule, and creates SpeciesGroup<2> objects for all of the bonds.
