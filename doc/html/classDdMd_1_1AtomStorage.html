Atom * DdMd::AtomStorage::newAtomPtr ( )

Returns pointer an address available for a new Atom.

This method returns the address of an Atom object that can be used for a new local Atom. It does not add the Atom the the atom set, and so must be followed by a call to addNewAtom() to do this.

Usage: 

 AtomStorage storage;
 Atom*       ptr;

 ptr = storage.newAtomPtr;
 ptr->setId(id);
 // Set other properties of the new Atom.
 storage.addNewAtom();
Returns:
address for new atom.

void DdMd::AtomStorage::addNewAtom ( )

Finalize addition of the most recent new atom.

This method adds the atom that was returned by the most recent call to newAtomPtr to the atom set. The global atom id must be set before calling this function, by calling Atom::setId(int), because the algorithm uses the global id returned by Atom::id().
