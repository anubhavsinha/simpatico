virtual void McMd::MdPairPotential::addForces ( ) [pure virtual]

Calculate non-bonded pair forces for all atoms in this System.

Adds non-bonded pair forces to the current values of the forces for all atoms in this system. Before calculating forces, this function checks if the pair list is current, and rebuilds it if necessary.

void McMd::MdPairPotential::buildPairList ( )

Build the internal PairList.

Before building the PairList, this method also shifts all atom positions into the primary image of the periodic boundary. In an MD simulation, the positions are allowed to wander out of the primary image between calls to this function.

bool McMd::MdPairPotential::isPairListCurrent ( )

Return true if PairList is current, false if obsolete.

A PairList is considered current iff it has been built previously, and no particle has moved by a distance greater than skin/2 since the PairList was last built.
