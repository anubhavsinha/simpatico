int DdMd::Domain::shift ( int  i,
int  j 
) const [inline]

Shift to be applied to sent coordinates in transfer (i, j).

When communicating an atomic position that is stored on the sending processsor as x = {x[0], x[1], x[2]}, the value of coordinate x[i] should be sent as x[i] - shift(i, j)*length[i] during transfer (i, j), where length[i] is the length of the unit cell in direction i.

Parameters:
iindex of Cartesian direction 0 <= i < Dimension
jindex for positive (j = 0) or negative (j = 1) transfer
