Detailed Description

Native / default format for configuration files.

DdMdConfigIo is a ConfigIo that implements the default configuration file format for ddSim.

Write configuration file.

This routine opens and writes a file on the master, collecting atom data from all processors. Atomic positions are written in Cartesian coordinates, independent of the coordinate system used by the AtomStorage.

Parameters:
fileoutput file stream (must be open on master)
