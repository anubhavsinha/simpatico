Read the parameter file block.

Inherited from ParamComponent. This function reads the entire parameter block for this ParamComposite, including the opening line "ClassName{" and the closing bracket "}". The default implementation calls the virtual readParameters function to read the body of the block, and adds Begin and End objects.
