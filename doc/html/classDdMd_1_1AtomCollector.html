DdMd::AtomCollector Class Reference
MPI Communication

Detailed Description

Class for collecting Atoms from processors to master processor.

An AtomCollector collects atoms from all processors to the master in order to output a configuration file.

Usage: 

    AtomStorage storage;
    Domain domain;
    Buffer buffer;
    AtomCollector collector;

    // Initialization
    collector.associate(domain, storage, buffer);

    // Communication
    if (domain.gridRank() == 0) {  // if master processor
       collector.setup();
       Atom* atomPtr = collector.nextPtr();
       while (atomPtr) {
          // Write *atomPtr to file; 
          atomPtr = collector.nextPtr();
       }
    } else { // if not master
       collector.send();
    }
