McMd::McEnergyPerturbation Class Reference
Free Energy Perturbation

Detailed Description

A Perturbation with a variable inverse temperature.

An McEnergyPerturbation is a series of systems with the same Hamiltonian at different temperatures. The perturbation parameter is beta = 1/kT.
