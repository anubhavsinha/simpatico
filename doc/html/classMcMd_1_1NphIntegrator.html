McMd::NphIntegrator Class Reference
Molecular Dynamics Integrators

Detailed Description

An explictly reversible/measure-preserving Parrinello-Rahman type NPH integrator.

TwoStepNPH can be used in three modes:

  • isotropic volume flucutations, cubic box
  • anisotropic volume fluctuations, orthorhombic box (three independent lengths)
  • anisotropic volume fluctuations, tetragonal box (two independent lengths, Ly = Lz)

The first mode is an implementation of the Andersen barostat, the second and the third mode are based on the adapted Parrinello-Rahman equations of motion.

The integration scheme used to implement the equations of motion is explicitly reversible and measure-preserving. It is based on the Trotter expansion technique introduced by Tuckerman et al. J. Chem. Phys. 97, pp. 1990 (1992).
