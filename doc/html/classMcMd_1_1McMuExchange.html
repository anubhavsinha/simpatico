McMd::McMuExchange Class Reference
MC Analyzers

Detailed Description

Exchange chemical potential for semigrand ensemble.

This class calculates the excess free energy cost of transforming a randomly chosen molecule of a specified species into another species by a hypothetical "alchemical" transformation of the atom types of some or all atoms in the molecule.
