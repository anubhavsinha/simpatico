RigidDisplaceMove

Synopsis

This mcMove attemps a rigid displacement of a randomly chosen molecule of a specified species.

See also:
McMd::RigidDisplaceMove

Parameters

The parameter file format is: 

   RigidDisplaceMove{ 
      probability        double
      speciesId          int
      delta              double
   }

in which

probability probability that this move will be chosen.
speciesId integer index of molecular species
delta maximum displacement
