void DdMd::Integrator::clear ( ) [virtual]

Set integrator to initial state and clears all statistics.

Clear timing, dynamical state, statistics, and diagnostic accumulators.

This method resset iStep = 0, calls initDynamicalState(), calls the DiagnosticManger::clear(), clears the internal Timer and all timing statistics, clear the additional timing statistics maintained by the Exchanger class, and clears the memory usage statistics for the Buffer, PairList, and storage classes.

void DdMd::Integrator::computeForces ( ) [protected]

Compute forces for all local atoms, with timing.

Identical to Simulation::forceCompute(), with added timing. Upon return, forces are correct for all local atoms. Values of the forces on ghost atoms are undefined. Executes reverse communication if needed, and emits Simulation::forceSignal().
