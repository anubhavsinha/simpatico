Detailed Description

Autocorrelation for vector separation of any two monomers on a molecule.

This class evaluates the autocorrelation function for the vector that separates two atoms with indices atom1Id and atom2Id on a molecule of species speciesId. It can be used to evaluate the autocorrelation function for individual bonds or for the end-to-end vector of a linear polymer.
