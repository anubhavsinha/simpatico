template<class Interaction >
void McMd::MdPairPotentialImpl< Interaction >::readParameters ( std::istream &  in)
virtual

Read pair potential interaction and pair list blocks.

This method reads the maxBoundary, PairList and pair potential Interaction parameter blocks, in that order, and initializes an internal PairList. Before calling the Interaction::readParameters method, it passes nAtomType to Interaction::setNAtomType().
