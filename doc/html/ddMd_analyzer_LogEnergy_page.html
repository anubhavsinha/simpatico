LogEnergy

Synopsis

This analyzer outputs the energy and its components to the log stream, which is set to standard output by default.

See also:
DdMd::LogEnergy

Parameters

The parameter file format is: 

  LogEnergy{
    interval           int
  }

with parameters

interval number of steps between data samples

Output

Instantaneous energy values are output to the log stream every interval steps, with all components in one line. Within each output line componentsappear in the order:

kinetic pair bond angle dihedral total

Unused components of the energy will not appear. A component of the energy may be unused either because it was not enabled at compile time (e.g., if the program was not compiled with INTER_ANGLE defined) or if was not enabled in the parameter file (e.g., if nAngleType == 0).
