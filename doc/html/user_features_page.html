These pages provide links to user documentation for features of each program that can be selected at run time by including appropriate polymorphic blocks in the parameter file to choose particular potential energies (interaction classes), molecular dynamics integrators or Monte Carlo moves, and data analysis operations. User documentation for each feature specifies the required file format for the associated parameter file block.
