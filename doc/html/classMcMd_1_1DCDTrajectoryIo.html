McMd::DCDTrajectoryIo Class Reference
Trajectory File IO

Detailed Description

TrajectoryIo for CHARMM ".dcd" data files.

Currently it has only been tested with Hoomd generated trajectory files.

It is assumed that the atoms belonging to all species and molecules of a species occur in the file consecutively.
