Util::MpiLoader< IArchive > Class Template Reference

Detailed Description

template<class IArchive>
class Util::MpiLoader< IArchive >

Methods to load data from input archive and broadcast to other processors.

Each of the "load" method templates causes the ioProcessor to load a variable from an archive and then broadcast it to all other processors in the communicator provided by an associated MpiFileIo object.
