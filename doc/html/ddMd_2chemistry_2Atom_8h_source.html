419  /*
420  * Accessors for pseudo-members stored in separate arrays.
421  *
422  * An atom can be constructed only as an element of a parent AtomArray
423  * (see documentation for private default constructor). Some data owned
424  * by an Atom is stored in a set of private array that are allocated by
425  * its AtomArray. In each such array, the element associated with this
426  * Atom is indexed by the localId_ of this atom, without the least
427  * signficant bit (see documentation of the localId_ member). The bit
428  * shift operation "localId_ >> 1" in each of the following functions
429  * strips off the least signficant bit, which is used to store the isGhost
430  * flag, and returns the relevant array index. Each arrays that stores
431  * pseudo-member data is a private member of the parent AtomArray (e.g.,
432  * AtomArray::velocities_, AtomArray::masks_, etc.) that is accessible
433  * because Atom is a friend class of AtomArray.
434  */
