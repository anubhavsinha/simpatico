267  /*
268  * Accessors for associated data stored in arrays allocated by AtomArray.
269  *
270  * An atom can be constructed only as an element of a parent AtomArray
271  * (see documentation for private default constructor). Some data owned
272  * by an Atom is stored in a set of private array that are allocated by
273  * its AtomArray. In each such array, the element associated with this
274  * Atom is indexed by the localId_ of this atom, without the isGhost
275  * flag. The bit shift operation "localId_ >> 1" in each of the following
276  * functions strips off the least signficant bit, which is used to store
277  * the isGhost flag, and returns the relevant array index. This array index
278  * is set for each atom when AtomArray is allocated, and is not changed
279  * thereafter.
280  */
