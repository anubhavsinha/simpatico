00474    * Accessors for pseudo-members stored in separate arrays.
00475    *
00476    * An atom can be constructed only as an element of a parent AtomArray
00477    * (see documentation for private default constructor). Some data owned
00478    * by an Atom is stored in a set of private array that are allocated by
00479    * its AtomArray. In each such array, the element associated with this 
00480    * Atom is indexed by the localId_ of this atom, without the least 
00481    * signficant bit (see documentation of the localId_ member). The bit 
00482    * shift operation "localId_ >> 1" in each of the following functions 
00483    * strips off the least signficant bit, which is used to store the isGhost

00485    * pseudo-member data is a private member of the parent AtomArray (e.g.,
00486    * AtomArray::velocities_, AtomArray::masks_, etc.) that is accessible
