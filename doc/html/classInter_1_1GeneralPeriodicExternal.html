Detailed Description

A clipped cosine potential that induces ordering along directions specified by waveIntVectors, w_i.

/ / / / w_i.(x-shifts_[0]) w_i.(y-shifts_[1]) w_i.(z-shifts_[2]) \ \ \ \ u = prefactor[atomType] externalParameter tanh | clipParameter| Sum Prefactor_[i] | cos | 2 pi ------------------ + ------------------ + ------------------ + phase_i | | | | \ \ i \ \ Lx Ly Lz / / / /

Prefactor (which depends on the atomType), externalParameter, waveIntVectors, interfaceWidth and periodicity are given as inputs in the parameter file. ClipParameter is the inverse of 2*pi*periodicity*interfaceWidth.
