Util::FPArray< Data, Capacity > Class Template Reference

Detailed Description

template<typename Data, int Capacity>
class Util::FPArray< Data, Capacity >

Statically allocated pointer array.

A FPArray is a statically allocated array that only holds pointers to Data objects, but for which the [] operator returns a reference to the associated data. It is implemented as a wrapper for a statically allocated C array of Data* pointers. A FPArray is not responsible for destroying the associated Data objects.

The interface of an FPArray is identical to that of an FSArray. An FPArray has both a capacity, which is the physical size of the underlying C array, and a size, which is the number of contiguous elements (from 0 to one less than size) that contain initialized data. The size can only be increased only by the append() method, which adds an element to the end of the array. When bounds checking is on, the operator [] checks that the index is less than the size.
