Constructor for throwing.

Constructs error message that includes file and line number. Values of the file and line parameters should be given by the built-in macros __FILE__ and __LINE__, respectively, in the calling function. A typical call of the constructor is thus of the form: 

    throw TestException("MyClass::myFunction", "A terrible thing happened!",
                    __FILE__, __LINE__ );
