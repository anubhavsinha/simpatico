void McMd::McPairPotentialImpl< Interaction >::readParameters ( std::istream &  in) [virtual]

Reads maxBoundary and pair potential Interaction blocks.

This method reads the maxBoundary and Interaction parameter blocks, and then allocates memory for an internal CellList. It passes simulation().nAtomType() as an argument Interaction::setNAtomType() before calling Interaction::readParameters().
