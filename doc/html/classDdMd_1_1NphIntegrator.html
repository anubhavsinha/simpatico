DdMd::NphIntegrator Class Reference
MD Integrators

Detailed Description

A reversible, symplectic NPH integrator (Anderson barostat).

Integration algorithm based on Martyna,Tobias and Klein (J. Chem. Phys. 101, 4177-4189, 1994).
