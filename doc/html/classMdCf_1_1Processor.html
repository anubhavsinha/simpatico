Read and analyze a sequence of numbered configuration files.

This function reads and analyzes a sequence of configuration files that were generated by running a previous simulation. The function reads files with names of the form inputPrefix() + n for integer suffixes min <= n <= max.

Parameters
mininteger suffix of first configuration file name
maxinteger suffix of last configuration file name
