Detailed Description

template<class BarePair, class LinkPotential>
class Inter::CompensatedPair< BarePair, LinkPotential >

Compensated pair potential for transient gel.

This class is intended for use in a model with transient crosslinks that can be created or destroyed at fixed chemical potential.
